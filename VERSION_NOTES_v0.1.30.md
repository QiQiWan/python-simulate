# v0.1.30 boundary preset and demo workflow update

## Highlights
- Added boundary condition preset library with one-click templates for pit models.
- Exposed preset application in the Stage/Boundary UI for both current stage and global model BCs.
- Improved default pre-solve guidance to point users to boundary presets instead of requiring manual target/component input.
- Made boundary/load target inputs editable combo boxes with common aliases prefilled.
- Marked the demo wall as `display-only` directly in the stage activation tree with a tooltip, reducing confusion when users try to activate it without interfaces.
- Updated the parametric demo metadata and example script to use v0.1.30 and the rigid-box boundary preset by default.
