# 0.8.71 - Production Meshing Validation

## Scope

This release adds a production meshing validation layer for 3D geotechnical workflows. It does not require Gmsh or meshio at import time; optional production Tet4 capability is reported through a health/status contract.

## Main additions

- `contracts.meshing_validation` dependency-light DTOs.
- `services.production_meshing_validation` headless service.
- Meshing/geotechnical module facade entries for STL repair, dependency health, region quality and interface conformity.
- Optional workflow `mesh_validation` typed quality artifact for STL/production meshing runs.
- Qt-free `MeshingValidationActionController`.
