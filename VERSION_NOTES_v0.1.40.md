# v0.1.40

- Fixed demo pit solver compatibility so the autogenerated crown beam uses translational-only truss2 elements instead of frame3d rotational DOFs. This keeps the autogenerated example solvable even when SciPy sparse is unavailable.
- Added dense fallback helpers for block sparse matrices and improved nonlinear structural hybrid assembly fallback paths.
- Added pre-solve diagnostics for active rotational structural elements when SciPy sparse is missing.
